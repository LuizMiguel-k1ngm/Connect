
## 📖 Sobre o Projeto

O **Connect** foi criado como uma página institucional para um grupo de jovens e adolescentes cristãos. Seu propósito é servir como uma porta de entrada para visitantes, reunindo informações importantes sobre o ministério, programação, valores e formas de participação.

A interface foi desenvolvida com foco em simplicidade, responsividade e uma experiência agradável para todos os dispositivos.

---

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla JS)**
